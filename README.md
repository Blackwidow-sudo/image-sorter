# ImageSorter

## Project install
```
python3 -m venv .venv
```

```
source .venv/bin/activate
```

```
pip3 install -r requirements.txt
```

## Running the app

Run the python src directly:

```
python3 main.py
```

Or build the app for MacOS:

```
./build.sh
```

Now you have a `ImageSorter.app` in the dist directory which you can install and use on MacOS.
